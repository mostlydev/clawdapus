# Shared Volume Contract

This directory represents the shared research and policy space.

In runtime, this is mounted from the `trading-shared` volume to `/mnt/trading-shared`.

Expected layout:

- `/mnt/trading-shared/skills`
- `/mnt/trading-shared/policy`
- `/mnt/trading-shared/research`
- `/mnt/trading-shared/reports`
- `/mnt/trading-shared/logs`
